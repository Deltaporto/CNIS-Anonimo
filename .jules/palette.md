## 2026-04-18 - Accessible Status Updates and Loading States\n**Learning:** Implementing `try...finally` blocks for UI loading states ensures buttons reset predictably even if async operations (like JSZip generation) fail. Adding `role="status"` and `aria-live="polite"` to status elements makes dynamic updates accessible to screen readers without interrupting the user.\n**Action:** Use `try...finally` for all blocking UI interactions and wrap dynamic feedback elements with ARIA live regions.
